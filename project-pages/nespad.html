The NesPad

What is the innovation we actually need?

Nespad – the notepad of the past.

A fully functional notepad with saving (WOW!) and 3 full fonts: normal, bold, and italic – all of which can be capitalized. AND you can also type all of the symbols used to make the UI.

And if that wasn’t enough for you modern folks, it even supports four different colors – though you can only change colors per line of text.

To this day the nespad remains one of my favorite projects, it was made for the course Retro Games and Emulator Programming given by Tom Tesch.
Whilst in the beginning it was a mess to wrap my head around assembly it quickly turned into a fun puzzle of saving every possible bit to squeeze out as much usable memory as we could.

For Example this is how we planned out the entirety of the saveable text.

8kb total battery backed ram
Each page has 256 bytes
byte 0–251 = text
bits from 252–255 (32 bits):
0–17 (18 bits) = color (2 bits per line, uses the same color system as the PPU)
18–26 (9 bits) = formatting bit per line (0 = normal, 1 = left formatting)
27–31 (5 bits) = document index

Side note for the color bits: the order in the bytes is a bit weird
xxxx xxxx xxxx xxxx nnnn nnxx
• If you look in memory, the x’s will be the color info
• First 2 bits are for line 3, second 2 for 2, then 1, and then 0
• Then in the second byte we have the first (leftmost) 2 for 7, then 6, then 5, then 4
• Rightmost bits of the third byte are assigned to line 8

Reserved byte in zero page to store what type of text we are writing (zp_text_info)
bit 1–2 = text type (00 = normal, 01 = bold, 10 = italic)
bit 3 = is capitalized
bit 4 = is on the special keyboard
bit 5–8 = used to store what line we are editing (should be reset when new page is loaded)

So as you can see we use every single bit we can to store information.
though its kind of a lie, since we dont actually use the document index.
The idea was to have different documents you could save seperately,
but this ended up being out of scope for the semester we had this course.
